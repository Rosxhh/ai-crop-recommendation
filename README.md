# AI Crop Recommendation System

This project recommends suitable crops based on soil nutrients and environmental conditions using Machine Learning.

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Flask/Django

## Features

* Crop prediction based on input data
* Machine learning model integration
* User-friendly interface

## Usage

1. Enter soil and environmental data
2. Get recommended crops instantly
